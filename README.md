# Ludoteca 🎲📚

Este es un proyecto fullstack que gestiona una ludoteca: juegos, autores, categorías, clientes y préstamos.
Está desarrollado con **Angular** en el frontend y **Spring Boot** en el backend.

---

## 🛠️ Tecnologías usadas

### Frontend (Angular)
- Angular CLI
- TypeScript
- SCSS
- Angular Material

### Backend (Spring Boot)
- Java 17+
- Spring Boot
- JPA / Hibernate
- H2
